# Scales by Genre

## Rock / Hard Rock
- **Pentatónica menor** — la más usada. E minor pentatonic: e|12-15| B|12-15| G|12-14| D|12-14| A|12-14| E|12-15|
- **Blues scale** — pentatónica menor + b5 (blue note)
- **Mixolidian** — para riffs tipo classic rock (AC/DC, ZZ Top)

## Blues
- **Blues scale** (A): e|5-8| B|5-8| G|5-7| D|5-7| A|5-7| E|5-8|
- Agrega b3→3 bends para el characteristic "cry"
- Posición en el 12th fret para sonar más vocal

## Psychedelic / Indie (Tame Impala style)
- **Menor natural** — frases espaciadas con mucho sustain
- **Dorian** — para un color más "cool" (raise b7→7)
- **Pentatónica menor** con bends suaves y vibrato lento
- Tempo en 8th notes con mucho espacio entre notas

## Metal
- **Frígio** — para riffs oscuros (b2 da el color característico)
- **Menor armónica** — para solos dramáticos
- **Pentatónica menor** + cromatic passing tones

## Jazz / Fusion
- **Dorian** — estándar para jazz-rock
- **Mixolidian** — sobre acordes dominantes
- **Altered scale** (7th mode de melodic minor) — para tensión sobre V7

## Funk / R&B
- **Pentatónica menor** con mucho énfasis rítmico, notas cortas
- **Mixolidian** para grooves más soul
- El ritmo importa más que las notas — 16th note feel

## Folk / Country
- **Mayor pentatónica** — bright, optimista
- **Mixolidian** — country twang
- Open string licks son característicos
